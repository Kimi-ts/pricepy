Инструкция по добавлению новости:

1. Сначала загружаешь изображение (Choose File кнопка выбора файла, затем нажимай Save Image).

2. Если на первом пункте всё прошло без ошибок, то в списке всех новостей первой появится новость с твоей загруженной картинкой. Эта новость будет выделена красной рамкой.

3. Слева находится панель, где ты можешь редактировать текст новости. Справа - то, как будет выглядеть новость на сайте (как только ты изменяешь текст слева - текст справа сразу же меняется). Обрати внимание, что для красоты текст новости должен быть заключён в специальные html-тэги. Изучи внимательно пример новостей, которые уже есть на сайте. Например, каждая новая строчка должна начинаться с '<'p'>'' и заканчиваться '<'/p'>. Заголовок должен начинаться с '<'h3 class=textBlock-subheadline'>' и заканчиваться '<'/h3'>' и т.д.
